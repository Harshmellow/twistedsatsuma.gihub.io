Parents suck but deal with them

Up to this point we have not talked about "Parent-elements". There not much different, there only real difference is that they hold other elements.

Think about Russian dolls, the biggest one holds all the other dolls.

In this case the parent <ol>
		holds or contains the child <li>

Difference between ol and ul

Another type of list is <ul> . Its a parent too! Which means that it controls what kind of list it is. <ul> creates bulleted lists

  • Bulleted lists
  • Still uses the child
  • li

Lets Learn the last piece TABLES!

Tables have 3 main components or tags

<table> <tr> and <td>

These things stand for different things

  • Table = Parent,holds tr,td elements
  • tr = Child, table-row,
  • td = Child, table-data
